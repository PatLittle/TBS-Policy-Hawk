# Changes to the Leave Reporting System - New Reporting Requirements

Date: October 31, 2012

To: Directors/Chiefs of Labour Relations and Compensation

This bulletin provides information regarding new reporting requirements for compensatory codes in the Leave Reporting System.

In order to properly track statistics on compensatory leave, it is necessary to add the following codes to the Treasury Board of Canada Secretariat's list of mandatory Leave Reporting System codes.

## Compensatory Leave

- Code 268 – Compensatory – Overtime while on training
- Code 290 – Compensatory – Overtime – Other rate

Departments should ensure that their leave reporting system is amended accordingly before March 31, 2013.

Questions regarding this bulletin should be directed to your departmental corporate compensation official.

Michael Holt A/Senior Director Core Public Administration Compensation Management Compensation and Labour Relations Office of the Chief Human Resources Officer
