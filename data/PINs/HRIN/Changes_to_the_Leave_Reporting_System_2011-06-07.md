# Changes to the Leave Reporting System - June 2011

Date: June 7,2011

To: Heads of Human Resources, Directors/Chiefs of Labour Relations and Compensation

The purpose of this bulletin is to inform departments of two changes that have been made to the Leave Reporting System (LRS).

## Code 665 - Quarantine Leave

The first change involves a new mandatory code being created for quarantine leave which was introduced into the Health Services (SH) collective agreement signed on May 31, 2005. Code 665 - Quarantine Leave has been created under Other Paid Leave.

## Code 680 - Special Leave for the HP Group

The second change is to disable code 680 – Special Leave for the HP Group, since there is no longer a special leave provision for employees in the Heating, Power and Stationary Plant Operations (HP) group in the Operational Services collective agreement.

All questions should be directed to your Departmental Corporate Labour Relations or Corporate Compensation Official who, if required, will direct their questions to the following email address: [Contact Interpretations by E-mail Interpretations@tbs-sct.gc.ca](mailto:Interpretations@tbs-sct.gc.ca)
